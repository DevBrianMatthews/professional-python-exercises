# 🐍 Professional Python Exercises

> Real-world programming challenges designed to sharpen your logic and prepare you for professional development.

These are not academic exercises. Every challenge is grounded in scenarios you will actually encounter as a developer — input validation, data processing, system design, error handling, and more.

---

## 🗂️ Structure

Challenges are organized by difficulty:

```
professional-python-exercises/
├── easy/
├── medium/
├── hard/
└── super-hard/
```

Each challenge lives in its own folder:

```
easy/
└── E01-password-validator/
    ├── challenge.md          ← Problem statement (EN + ES)
    └── solutions/
        └── python/
            ├── brian_matthews.py
            └── your_github_username.py
```

---

## 📊 Difficulty Levels

| Level        | Description                                            |
| ------------ | ------------------------------------------------------ |
| `easy`       | Core Python, basic logic, single functions             |
| `medium`     | OOP, error handling, data structures                   |
| `hard`       | Algorithms, testing, system design                     |
| `super-hard` | Complex architecture, optimization, real-world systems |
