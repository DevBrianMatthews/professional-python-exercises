# Contributing

## How to submit your solution

1. **Fork** this repository
2. **Clone** your fork locally
   ```bash
   git clone git@github.com:your_github_username/professional-python-exercises.git
   ```
3. Add your solution file inside the challenge folder, under `solutions/python/`:
   ```
   easy/E01-password-validator/solutions/python/your_github_username.py
   ```
4. **Commit** your change
   ```bash
   git add .
   git commit -m "#E01 - Python"
   ```
5. **Push** and open a **Pull Request** against `main`

---

## Rules

- One file per person per challenge
- File name must be your GitHub username in lowercase: `your_github_username.py`
- Do not modify any other files
- PR title format: `#E01 - Python` (challenge number + language)
- PRs that do not follow this format will be rejected

---

## Notes

- Any language is welcome — if there is no folder for your language yet, create it
- You can submit solutions in multiple languages (one PR per language)
- Constructive feedback on other solutions via PR comments is encouraged
